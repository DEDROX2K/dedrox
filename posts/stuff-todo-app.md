# Stuff Todo App


1. Get task dependencies
		Set-up dependencies between tasks to determine what needs to be done first and what can come later
2. Browse the adventure log
		Browse, filter and review tasks that you previously completed along with any daily notes
3. Add tasks with listen mode
4. Scan task to add to stuff
5. Organize with tags
		Use tags and pins to organize tasks
6. List templates
7. Cancel task, list and notes
8. Set deadlines
