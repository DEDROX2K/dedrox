How you control cursor per element now:

1. In HTML: set `data-cursor="default|text|link|grab|grabbing"` on any element.
2. In JS at runtime:
```js
CursorControl.setRole('.my-card', 'link');
CursorControl.clearRole('.my-card');
CursorControl.setCursorAsset('link', 'images/my-link-cursor.png', 6, 4, 'pointer');
```

Why this is safer:
- Cursor logic is centralized under `body.cursor-system` and uses `!important` in one policy block.
- Scattered hardcoded custom-cursor declarations were removed from components, so style edits elsewhere won’t silently override your cursor behavior.
